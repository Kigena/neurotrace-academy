# Complete Pattern Image Naming Guide

This guide lists ALL patterns in the system and their corresponding image file names.

## How to Use This Guide

1. Save your EEG images to: `public/images/patterns/`
2. Name them EXACTLY as listed below (case-sensitive)
3. Images will automatically appear on pattern detail pages

---

## Normal Patterns

| Pattern Name | Image File Name |
|--------------|----------------|
| Posterior Dominant Rhythm (PDR) | `pdr.png` |
| Sleep Spindles | `sleep-spindles.png` |

## Special Cases

### SREDA (Multiple Images)
SREDA supports multiple images. Name them:
- `sreda-1.png`
- `sreda-2.png`
- `sreda-3.png`
- `sreda-4.png`

All will display on the SREDA pattern page.

---

## Image Requirements

- **Format:** PNG (preferred) or JPG
- **Size:** 1200-2000px width recommended
- **File Size:** Keep under 2MB for fast loading
- **Location:** `public/images/patterns/`

---

## After Adding Images

1. Images automatically appear on pattern detail pages
2. Refresh your browser to see them
3. Images display in both "Core Identity" and "ABRET Pearls" tabs
