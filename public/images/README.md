# NeuroTrace Academy - Learning Images

This directory contains educational images for the NeuroTrace Academy platform.

## Current Images

### Trace Alternant PSG Pattern
- **File:** `trace-alternant-psg.png`
- **Description:** Polysomnography (PSG) recording showing trace alternant pattern in neonatal quiet sleep
- **Used in:** Domain I - Age-Related EEG Development section
- **Key Features:**
  - Shows alternating attenuation (purple boxes) and activity (orange boxes) periods
  - Demonstrates normal discontinuous pattern in neonates
  - Includes EEG, EOG, EMG, respiratory, and ECG channels
  - Illustrates normal quiet sleep state in neonates

## Adding New Images

When adding new images:
1. Place image files in this directory
2. Use descriptive, lowercase filenames with hyphens (e.g., `pattern-name-description.png`)
3. Update this README with image details
4. Reference images in components using `/images/filename.png`

## Image Guidelines

- **Format:** PNG or JPG
- **Naming:** Use descriptive names with hyphens (e.g., `trace-alternant-neonatal-sleep.png`)
- **Size:** Optimize for web (recommended max 2MB per image)
- **Accessibility:** Always include descriptive alt text in components
