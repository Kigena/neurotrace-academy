# Neuroanatomy Images Directory

This directory contains anatomical brain images used in the NeuroTrace Academy study guides.

## Image Files Required

### 1. brain-lateral-detailed-gyri-cranial-nerves.png
**Description:** Detailed lateral view of the brain showing:
- All major gyri (precentral, postcentral, superior/inferior frontal, superior/middle/inferior temporal)
- All major sulci (central sulcus, lateral fissure/Sylvian fissure, intraparietal sulcus)
- Pars opercularis, triangularis, orbitalis
- Supramarginal gyrus, angular gyrus
- Cerebellum structures (primary fissure, horizontal fissure, flocculus)
- Cranial nerves (Trigeminal, Abducent, Hypoglossal, Vestibulocochlear, Facial, Glossopharyngeal, Vagus)

**Used in:**
- `Domain-I/D1-12-Neuroanatomy-Brain-Structures.md` (Section 2: External Structures)

### 2. brain-lateral-colored-gyri.png
**Description:** Lateral view of the brain with color-coded functional areas:
- Precentral gyrus highlighted in RED (primary motor cortex)
- Postcentral gyrus highlighted in BLUE (primary sensory cortex)
- Central sulcus clearly marked
- Four major lobes labeled (Frontal, Parietal, Temporal, Occipital)
- Lateral sulcus (Sylvian fissure)
- Cerebellum and Brainstem (Pons, Medulla)

**Used in:**
- `Domain-I/D1-12-Neuroanatomy-Brain-Structures.md` (Section 2: Frontal Lobe, Parietal Lobe)
- `Domain-I/D1-05-Neuroanatomy-EEG-Localization.md` (Section 2: Major Cortical Lobes)
- `Domain-II/D2-16-Electrode-Placement-Complete-Guide.md` (Section 2: Standard 10-20 System)

### 3. circle-of-willis-inferior-view.png
**Description:** Circle of Willis from inferior (bottom) view showing:
- Complete arterial anastomotic ring at base of brain
- Internal Carotid Arteries (ICA)
- Anterior Cerebral Arteries (ACA)
- Middle Cerebral Arteries (MCA)
- Anterior Communicating Artery (ACOM)
- Posterior Communicating Arteries (PCOM)
- Posterior Cerebral Arteries (PCA)
- Basilar Artery
- Vertebral Arteries
- Cerebellar arteries (PICA, AICA, SCA)
- Brain structures visible in light gray/pink

**Used in:**
- `Domain-I/D1-12-Neuroanatomy-Brain-Structures.md` (Section 6: Blood Supply Summary)

### 4. circle-of-willis-educational.png
**Description:** Educational slide/diagram showing:
- Textual breakdown of Circle of Willis components
- Anterior circulation (ICA, ACA A1 segments, ACOM)
- Posterior circulation (PCOM, PCA P1 segments, Basilar tip)
- Stylized diagram of arterial network
- All major arteries labeled
- Cerebellar arteries (PICA, AICA, SCA)
- Vertebral arteries

**Used in:**
- `Domain-I/D1-12-Neuroanatomy-Brain-Structures.md` (Section 6: Blood Supply Summary)

### 5. arteries-brain-inferior-view.png
**Description:** Simplified diagram of brain arteries from inferior view:
- Red arterial network on light gray brain outline
- Circle of Willis clearly visible
- Major arteries labeled with connecting lines
- Cerebellar arteries shown
- Source citation visible

**Used in:**
- `Domain-I/D1-12-Neuroanatomy-Brain-Structures.md` (Section 6: Blood Supply Summary)

### 6. aca-segments-medial-view.png
**Description:** Medial (sagittal) view showing Anterior Cerebral Artery:
- ACA segments A1 through A5 clearly labeled
- Course around corpus callosum
- Corpus callosum color-coded (precallosal green, main body yellow, postcallosal orange)
- Callosomarginal artery and supracallosal branches
- Connection to ICA and MCA
- ACOM shown in blue
- Brain structures in light pink/beige

**Used in:**
- `Domain-I/D1-12-Neuroanatomy-Brain-Structures.md` (Section 6: Blood Supply Summary - ACA section)

## Image Specifications

- **Format:** PNG or JPG
- **Recommended size:** 800-1200px width for web display
- **Quality:** High resolution for clarity
- **Background:** White, black, or transparent preferred

## Adding Images

1. Save the image files in this directory (`public/images/neuroanatomy/`)
2. Use the exact file names listed above
3. The study guides already have image references embedded - they will automatically display once the files are added
